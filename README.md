# Student-Record-Management-System-_-C
## Overview
- The Student Record Management System is a C language project designed to manage student academic records efficiently.
- It allows users to add, display, and delete student data, while automatically calculating average marks.
- The program also ensures input validation, preventing incorrect or invalid data entries.

## Features
- Add new student records (ID, Name, Marks for 5 subjects)
- Display all stored student records in a tabular format
- Compute average marks for each student automatically
- Validate names to contain only alphabetic characters
-  Delete a student record by name
-   Prevent overflow with a limit of 15 students
-    Menu-driven and easy-to-use interface

## Technologies Used

- Language: C Programming

# Concepts:
- Structures
- Arrays
- Functions
- Control Statements
- String Manipulation (string.h)
- Input Validation (ctype.h)
  
## Code Highlights

- Structure-based data storage for efficient handling of student information
- Error checking for numeric and alphabetic input types
